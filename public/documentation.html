This app allows user to create, edit, delete flashcards contained in relative class names.

  • To Create a User - '/users '
  • To Create Courses - '/courses/:userid'
  • To Create Flashcards - '/flashcards/:courseid'
  • To View List of Users - '/users'
  • To View User information by username - '/users/username/:username'
  • To View User information by id - '/users/id/:id'
  • To View List of Courses - '/courses/:userid'
  • To View Flashcards - '/flashcards/:courseid'
  • To Edit User Information - '/users/:username'
  • To Edit Course Name - '//courses/:courseid'
  • To Edit Flashcards - '/flashcards/:cardid'
  • To Delete User - '/users/:userid'
  • To Delete Courses - '/courses/:courseid'
  • To Delete Flashcards - '/flashcards/:courseid'
