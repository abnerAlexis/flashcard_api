This app allows user to create, edit, delete flashcards contained in relative class names.

  • To Create a User - '/sign-up'
  • To Create Courses - '/user/courses/create'
  • To Create Flashcards - '/user/courses/flashcards/create'
  • To View List of Users - '/users'
  • To View User information - '/users/username'
  • To View List of Courses - '/courses/view'
  • To View A single Course - '/courses/courseid'
  • To View Flashcards - '/courseid/flashcards/view'
  • To Edit User Information - 'users/userid/edit'
  • To Edit Courses - '/users/userid/courses/edit'
  • To Edit Flashcards - '/courseid/flashcards/edit'
  • To Delete User - '/profile/delete'
  • To Delete Courses - '/courses/courseid/delete'
  • To Delete Flashcards - '/flashcards/flashcardid/delete'
